## 4. v1 正式权益清单规则

进入 v1 正式权益清单需要同时满足：

1. 基金类型属于首期支持范围：股票型、混合型-偏股、混合型-灵活、指数型-股票。
2. 经过 canonical code 清洗，不使用空壳/错误代码。
3. NAV、持仓、行业、经理、费率、规模、权益仓位等基础字段齐全。
4. 最新一期股票持仓总权重 ≥ 50%，或已完成 ETF 联接穿透。
